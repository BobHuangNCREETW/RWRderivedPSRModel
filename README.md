# RWRderivedPSRModel
This study presents a new empirical probability model for surface rupture (PSR) for crustal earthquakes, a key component of probabilistic fault-displacement hazard analyses. The dataset is based on a literature review of earthquake source parameters from 1970 to 2023. The new PSR model uses the rupture-width ratio (RWR) instead of moment magnitude and styles of faulting (SoF), unlike most PSR models. The RWR is determined by dividing the rupture width ($W_{Rup}$) by the fault width. If the rupture dimension is unavailable, we assumed source-scaling models (SSM) from Huang et al. (SRL, 2024) (Hea24) that account for the width-limited effect based on regional seismogenic thickness to estimate $W_{Rup}$. The proposed PSR model uses the Beta cumulative distribution function (CDF), which incorporates two key physical factors: the influence of fault geometry on RWR and the localization of rupture at depth. The functional form derived from the Beta CDF rather than a typical logistic regression avoids unphysical trends arising from a sparse subset of earthquakes. Other physical indices, such as SoF, underground material stiffness, and a longer fault length indicated by the fault aspect ratio, were evaluated but not modeled due to high epistemic uncertainty resulting from limited earthquakes in certain subgroups. A recommended procedure is built to translate Hea24 from a regional scale into a fault-based evaluation method, which consists of the mean prediction and aleatory variability propagated from SSM to PSR in this study. Finally, the proposed PSR model accounts for fault-specific fault geometry and normalized average hypocenter depth, while incorporating empirically constrained forms that are better suited to fault-based PSR evaluation.


The new RWR-derived PSR model is provided in [sub_RWR_derived_PSRmodel.m], with an example main program in [main_prediction.m]. The regression code for an example table is provided in [sub_fitBetaMLE_fixbeta.m] for the stage 1 (S1) model and in [sub_fitBetaMLE_fixbeta_addedNMZHypo.m] for the stage 2 (S2) model, with an example main program in [main_regression.m].
